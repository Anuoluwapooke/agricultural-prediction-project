# Agricultural Yield Prediction Project

## Overview
This project analyses agricultural yield data using SQL and Python. The objective was to explore rainfall and crop yield patterns and build a regression model to predict agricultural risk.

## Tools Used
- SQL (SQLite) – Data querying and aggregation  
- Python (Pandas, Scikit-learn) – Data analysis and regression modelling  
- Matplotlib – Data visualisation

  
 ## Floder Structure

## Key Analysis
- Calculated average rainfall and yield per crop using SQL  
- Analysed yearly yield trends  
- Built a linear regression model to predict crop yield based on rainfall  
- Visualised yearly agricultural performance  

## Workflow
CSV Dataset → SQL Analysis → Python Modelling → Visualization

## Outcome
This project demonstrates end-to-end data analytics skills including structured querying, statistical modelling, and data visualisation.
